# Book Referrer | 今日宜读

> 给有翻阅一本书的念头却不知读什么的人，一份来自天气与心情的温柔指引。

**https://book-referrer.vercel.app/** 是一个感性而宁静的书籍推荐网页。

它不只是工具，更像一座小小的纸上驿站——根据你此刻的心情与窗外的天气，从豆瓣图书 Top 250 中选出一本或许正适合你的书。最后，所有心意会被打印成一张复古小票，尺寸恰好是手机壁纸的模样，可以保存，也可以分享。

## 🔖 核心特性

- **感性推荐**  
  从 **10 种心情** 与 **5 种天气** 出发，让一本书与你在此刻安静共鸣。

- **复古小票设计**  
  模拟打字机的印记，搭配莎草纸般的肌理与撕裂的锯齿边缘，回顾从旧时光里扯来的一纸书缘。

- **名言补全，从不缺席**  
  若书籍简介暂时留白，系统会从关于阅读、灵魂与生活的名人语库里，挑选一句恰好的话，温柔填补。

- **三款配色，三种心境**  
  - **静谧**：深灰底色（38,38,38）配矢车菊蓝（100,149,237）——如深夜与月光。  
  - **专注**：米黄底色（255,239,200）配深灰（38,38,38）——如旧书页的温厚。  
  - **沉思**：深蓝底色（35,48,115）配浅蓝（204,237,255）——如薄暮时分的湖面。

- **壁纸级导出**  
  一键生成 3 倍高清 PNG，完美适配手机屏幕尺寸，便于保存或分享给同样需要一本书的人。

## 🎨 设计哲学

项目遵循 **知性、田园、典雅** 的气质。

- **字体**：中文采用思源宋体（Noto Serif SC），英文与数字采用打字机字体（Special Elite），呈现跨越纸张与屏幕的文学质感。
- **排版**：以书名与简介的换行方式致敬收据的紧凑与真诚。
- **交互**：从“我想认识的下一本书”开始，每一个微动效都稳重、克制，给予引导与回应，却不过分。

## 🛠️ 技术实现

- **前端**：React + Tailwind CSS + Lucide Icons（轻量 CDN 引入）
- **数据处理**：Python（Pandas）预处理 Excel，实现心情与天气的智能归类
- **图像生成**：html-to-image 高清渲染
- **动效**：canvas-confetti + CSS3 关键帧动画

## 📖 本地翻阅

若想在本地查看或修改本项目，请依以下步骤操作：

1. 克隆本仓库  

2. 在项目根目录下启动本地服务（推荐 Python）：
   ```bash
   python3 -m http.server 8000
   ```

3. 浏览器访问 `http://localhost:8000`  
   内视一种心情，外观一眼天气，等一张属于你的小票，邂逅一本属于此刻的书。
