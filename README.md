# Book Referrer | 书籍推荐小票生成器

这是一个为“想要看书但不知道读什么”的人设计的感性书籍推荐网页。它不仅提供阅读指引，更通过复古的打字机质感和典雅的美学设计，为用户开启一段充满仪式感的阅读邂逅。

## 😎 核心特性

- **感性推荐算法**：根据用户当前选择的 **10 种心情**（如愉悦、思念、迷茫等）和 **5 种天气**，从豆瓣图书 Top 250 中智能筛选最契合氛围的书籍。
- **复古小票设计**：生成一张模拟打字机印迹、带有莎草纸纹理和撕裂锯齿边缘的“书籍小票”。
- **精美名言补全**：若书籍简介缺失，系统会自动从名人名言库中选取关于读书、灵魂与生活的文字进行艺术化补全。
- **三款典雅配色**：
  - **静谧**：深夜般的深蓝底色，搭配如月光般的白兰色细节。
  - **专注**：经典的莎草纸米黄质感，知性而稳重。
  - **沉思**：忧郁而深邃的矢车菊蓝，呈现冷静的文学美感。
- **壁纸级导出**：支持一键生成 3 倍高清 PNG 图片，完美适配手机壁纸尺寸，方便保存与分享。

## ☺️ 设计哲学

项目遵循 **“宁静、田园、典雅”** 的个性：
- **字体**：中文采用思源宋体（Noto Serif SC），英文与数字采用打字机字体（Special Elite），呈现跨越时代的文学质感。
- **排版**：使用书名与简介换行规则模拟真实收据的紧凑美学。
- **交互**：从“我想认识的下一本书”开始，每一处微动效都力求稳重、克制。

## 🛠️ 技术实现

- **前端**：React + Tailwind CSS + Lucide Icons (CDN 引入，轻量便捷)。
- **数据处理**：Python (Pandas) 预处理 Excel 数据，实现智能标签归类。
- **图像生成**：`html-to-image` 高清渲染技术。
- **动效**：`canvas-confetti` + CSS3 Keyframe 动画。

## 🚀 快速开始

1. 克隆仓库。
2. 在项目根目录下启动本地服务器（推荐使用 Python）：
   ```bash
   python3 -m http.server 8000
   ```
3. 在浏览器访问 `http://localhost:8000` 开启您的阅读邂逅。
